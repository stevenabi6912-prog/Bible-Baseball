# Sound Files

Place the following MP3 files in this directory:

- `organ-loop.mp3` — Old-timey baseball organ background music (looping)
- `crowd-cheer.mp3` — Crowd cheering (correct answer)
- `crowd-groan.mp3` — Crowd groaning (wrong answer / out)
- `homerun-fanfare.mp3` — Big cheer + fanfare (home run)
- `crowd-react.mp3` — Short crowd reaction (runner advances)
- `victory-jingle.mp3` — Victory jingle (game won)
- `defeat-jingle.mp3` — Defeat jingle (game lost)

All sounds should be royalty-free. Recommended sources:
- freesound.org
- mixkit.co
- pixabay.com/sound-effects

The app will work fine without these files — missing sounds are silently ignored.
